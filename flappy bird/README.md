# Flappy Bird Game

A simple implementation of the classic Flappy Bird game using HTML5 Canvas and JavaScript.

## How to Run

1. Clone or download this repository
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge, etc.)
3. That's it! No installation or special software required.

## How to Play

- **Objective**: Navigate the bird through the gaps between pipes without hitting them or the ground.
- **Controls**:
  - Press **Space** key to flap (on desktop)
  - **Tap** or **Click** the screen to flap (on mobile or desktop)
- **Scoring**: You earn one point for each pipe you successfully pass through.

## Game Features

- Simple, colorful graphics
- Responsive design that works on both desktop and mobile devices
- Score tracking
- Game over screen with final score and restart option
- Start screen with instructions

## Technical Details

This game is built using:
- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Canvas for rendering

No external libraries or frameworks are used, making it extremely portable and lightweight.

## Browser Compatibility

The game works on all modern browsers that support HTML5 Canvas:
- Chrome
- Firefox
- Safari
- Edge
- Opera
- Mobile browsers (iOS Safari, Android Chrome, etc.)
